# Markdown语法基础教程

Markdown是一种轻量级标记语言，创建于2004年，如今已成为世界上最受欢迎的标记语言之一。它允许人们使用易读易写的纯文本格式编写文档，并且可以转换成HTML等多种格式。本文将介绍Markdown的基本语法和使用方法。

## 为什么使用Markdown？

Markdown有以下几个主要优点：

- **易于学习和使用** - 语法简单，几分钟即可掌握基础
- **纯文本，易于阅读** - 即使不转换为HTML，原文也非常清晰易读
- **跨平台** - 可以在任何文本编辑器中编写
- **专注内容而非格式** - 让你专注于写作本身
- **广泛支持** - GitHub、Stack Overflow、Reddit等众多网站和工具支持Markdown

## 基本语法

### 标题

Markdown中的标题使用`#`符号，不同数量的`#`表示不同级别的标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 强调

```markdown
*斜体文本* 或 _斜体文本_
**粗体文本** 或 __粗体文本__
***粗斜体文本*** 或 ___粗斜体文本___
```

效果:
*斜体文本*
**粗体文本**
***粗斜体文本***

### 列表

无序列表使用`-`、`*`或`+`作为列表标记：

```markdown
- 第一项
- 第二项
- 第三项

* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项
```

有序列表使用数字加点：

```markdown
1. 第一项
2. 第二项
3. 第三项
```

### 链接

```markdown
[链接文字](URL "可选标题")
```

例如：[GitHub](https://github.com "GitHub官网")

### 图片

```markdown
![替代文本](图片URL "可选标题")
```

### 引用

使用`>`符号表示引用：

```markdown
> 这是一个引用。
> 
> 这是引用的第二段。
```

效果：
> 这是一个引用。
> 
> 这是引用的第二段。

### 代码

行内代码使用反引号(`)：

```markdown
使用`console.log()`打印日志
```

代码块使用三个反引号(```)：

````markdown
```javascript
function hello() {
    console.log("Hello, World!");
}
```
````

### 水平线

使用三个或更多的连字符、星号或下划线：

```markdown
---
***
___
```

### 表格

```markdown
| 表头1 | 表头2 | 表头3 |
| ----- | ----- | ----- |
| 单元格1,1 | 单元格1,2 | 单元格1,3 |
| 单元格2,1 | 单元格2,2 | 单元格2,3 |
```

效果：

| 表头1 | 表头2 | 表头3 |
| ----- | ----- | ----- |
| 单元格1,1 | 单元格1,2 | 单元格1,3 |
| 单元格2,1 | 单元格2,2 | 单元格2,3 |

## 扩展语法

一些Markdown解析器支持以下扩展语法：

### 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

### 数学公式

```markdown
行内公式：$E = mc^2$

块级公式：
$$
\frac{n!}{k!(n-k)!} = \binom{n}{k}
$$
```

### 脚注

```markdown
这里有一个脚注[^1]。

[^1]: 这是脚注的内容。
```
