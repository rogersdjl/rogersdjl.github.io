# 第一篇博客文章

欢迎来到我的博客！这是第一篇文章，我将介绍这个博客的一些特性和功能。

## 博客功能

这个博客基于纯HTML、CSS和JavaScript构建，采用小清新风格设计，主要功能包括：

1. **Markdown渲染** - 所有文章都使用Markdown格式编写
2. **代码高亮** - 支持多种编程语言的代码高亮显示
3. **数学公式** - 支持LaTeX数学公式渲染
4. **访问统计** - 记录网站和文章的访问量
5. **评论系统** - 允许读者发表评论

## 代码示例

下面是一个简单的JavaScript代码示例：

```javascript
function hello() {
    console.log("Hello, World!");
    return 42;
}

// 调用函数
const answer = hello();
console.log(`The answer is: ${answer}`);
```

## 数学公式示例

这个博客还支持数学公式渲染，例如：

行内公式：$E = mc^2$

行间公式：

$$
\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}
$$

## 如何添加新文章

添加新文章非常简单，只需：

1. 在`posts`目录中创建新的Markdown文件
2. 在`js/main.js`文件中的`posts`数组中添加新文章的信息
3. 完成！新文章会自动显示在博客中
